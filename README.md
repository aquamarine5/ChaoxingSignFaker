> [!NOTE]
> ChaoxingSignFaker（随地大小签）需要Android 8.0+（`minSdk>=26`）版本。

## 功能

- 自定义位置签到
- 自选择拍照签到
- 二维码签到（需要一台设备在现场扫码，理论上可为任意多的用户签到）
- 一台设备登录多账号以签到

> [!TIP]
> ChaoxingSignFaker 使用 [Stackbricks](https://github.com/aquamarine5/Stackbricks) 来实现更新逻辑，同样由作者维护。
